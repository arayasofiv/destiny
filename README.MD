Instrucciones para levantar la pagina:

Que se necesita? 
                -Xampp
                -PHP
                - y nada mas :D

Pasos a seguir:
                1- Instalar xampp, abrirlo y activar MySQL y tambien Apache.
                2- Meter la carpeta "Destiny" en el disco C, despues entras a xampp y por ultimo htdocs (y pones adentro de htdocs la carpeta) 
                3- Ya teniendo la carpeta ahi vas al navegador y pones localhost/destiny/ ahi entrarias a la pagina peero no se te carga la base de datos.
                4- Para cargarla tenes que poner localhost solo y al entrar apretas phpmyadmin, de ahi adentro del proyecto hay una carpeta llamada "db" que al entrar hay un archivo .SQL.
                5- En la pagina phpmyadmin creas la base de datos con nombre "destiny_shop".
                6- Ya creada esta, agarras el archivo .SQL mencionado anteriormente y vas a la parte de importar en el phpmyadmin y lo subis, le das continuar y ya estaria :DDD.
                7- Disfruta de la pagina!
